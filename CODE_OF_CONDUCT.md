# Code of Conduct

This project is an academic, preregistered research artifact released for the
purposes of transparency, reproducibility, and independent verification.

All participants and users are expected to engage in good faith, respect
scientific disagreement, and avoid personal attacks, harassment, or misuse of
the material.

This codebase is provided without warranty and is intended for research use.
Questions, critiques, and replication attempts are welcome when conducted
professionally and with reference to the associated preregistration.
